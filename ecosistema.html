Ecosistema Visual Inicio · Ecosistema Visual

La infraestructura
que alimenta la
inteligencia.

El Ecosistema Visual conecta datos, procesos, conocimiento experto e inteligencia artificial para generar capacidades de decisión especializadas para el sector agroalimentario.

TL;DR · En pocas palabras

El Ecosistema Visual son 8 productos integrados que capturan, conectan, contextualizan y activan información del campo: Visual, Agrovrain, MCP Visual, Agentes IA, FertiPRO, Selphi Control Horario, Visual Sensor y API.

Sobre esa infraestructura se activan las 6 Capacidades (Agronómica, Operativa, Climática y Territorial, Cumplimiento, Sostenibilidad y Varietal). La inteligencia no surge de un único producto: se construye conectando todos.

01 — Visual

La plataforma central de gestión e inteligencia agroalimentaria.

El núcleo del ecosistema. Centraliza toda la información agronómica (parcelas, tratamientos, riegos, fertilizaciones, partes de trabajo, costes) y la pone a disposición del resto de productos. Online, offline y multiplataforma.

Características

  • Cuaderno de Campo Digital
  • SIGPAC automático con intersección anual
  • Mapas de decisión multicapa
  • Geoportal GIS integrado (OGC, WMS, WFS)
  • App iOS y Android, web
  • Sincronización online y offline

Para qué sirve

  • Centralizar el dato agronómico
  • Coordinar equipos en campo
  • Cumplir cuaderno SIEX
  • Soportar todas las capacidades

Integra con

  • Agrovrain (IA agronómica)
  • MCP Visual (contexto)
  • FertiPRO (fertilización)
  • Visual Sensor (IoT)
  • API (ERP, CRM, BI)

02 — Agrovrain

Motor de inteligencia artificial especializado en agricultura.

El cerebro analítico del Ecosistema. Modelos predictivos entrenados con datos reales de campo: clima, satélite, plagas, ventanas óptimas de tratamiento, fenología y rendimiento. Le da capacidad de anticipación al resto de productos.

Capacidades del motor

  • Previsión agroclimática hiperlocal
  • Modelos de plagas y enfermedades
  • Predicción de cosecha y calibre
  • Recomendación de riego
  • Ventanas óptimas de tratamiento
  • Evaluación de riesgo climático

Fuentes de datos

  • +2.000 estaciones climáticas
  • Satélite Sentinel (Copernicus ESA)
  • Históricos agronómicos del cliente
  • Sensores físicos y virtuales
  • 4M+ ha de contexto

Capacidades alimentadas

  • Agronómica
  • Climática y Territorial
  • Sostenibilidad

03 — MCP Visual

La infraestructura contextual que conecta datos, conocimiento y modelos.

El plano de contexto que comparten todos los productos del Ecosistema. Conecta cada dato con su parcela, su histórico, su variedad y su normativa para que cuando un modelo de IA tome una decisión, lo haga con todo el contexto agronómico encima de la mesa.

Qué conecta

  • Datos operativos (Visual)
  • Modelos predictivos (Agrovrain)
  • Sensores IoT (Visual Sensor)
  • Sistemas externos (API)
  • Conocimiento agronómico estructurado

Para qué sirve

  • Dar contexto a los modelos de IA
  • Compartir núcleo de datos entre productos
  • Evitar silos internos
  • Acelerar nuevas capacidades

Beneficio para el cliente

  • Decisiones con contexto completo
  • Sin duplicar entradas de datos
  • Coherencia entre productos
  • Activación rápida de nuevas capacidades
Conocer MCP Visual →
04 — Agentes IA

Automatización especializada para procesos agroalimentarios.

Agentes inteligentes que ejecutan tareas recurrentes con criterio agronómico: detectar incidencias en imágenes, generar informes regulatorios, calcular dosis, planificar rutas o lanzar alertas. Cada agente está especializado en una tarea y aprende del contexto del cliente.

Tipos de agentes

  • Agente de cumplimiento normativo
  • Agente de detección de plagas
  • Agente de cálculo de dosis
  • Agente de planificación de rutas
  • Agente de evaluación de daños
  • Agente de reporting CSRD

Para qué sirve

  • Automatizar tareas repetitivas
  • Validar datos a escala
  • Liberar tiempo del agrónomo
  • Garantizar consistencia regulatoria

Operan sobre

  • Datos del MCP Visual
  • Modelos de Agrovrain
  • Imágenes de Selphi y satélite
  • Telemetría de Visual Sensor
Conocer Agentes IA →
05 — FertiPRO

Inteligencia aplicada a la fertilización.

Calcula la dosis exacta de fertilización por parcela integrando análisis de suelo, agua y materia orgánica. Genera los informes conformes al RD 1051/2022 listos para presentar, con trazabilidad completa vinculada al cuaderno.

Características

  • Cálculo de dosis por parcela
  • Integración análisis suelo y agua
  • Plan de fertilización RD 1051/2022
  • Libro de fertilizantes
  • Trazabilidad de aplicación
  • Reporte a regulador en un clic

Para qué sirve

  • Cumplir el RD 1051/2022
  • Optimizar coste de fertilizantes
  • Reducir huella ambiental
  • Auditar la aplicación a posteriori

Integra con

  • Visual (cuaderno de campo)
  • Sistemas de fertirrigación
  • MCP Visual (contexto)
  • ERP (compras de insumos)

06 — Selphi Control Horario

Gestión digital de equipos, jornadas y operaciones.

Resuelve el registro horario obligatorio para el trabajador de campo, el técnico en ruta y el personal de oficina. Móvil, tablet o web, con o sin cobertura. Verificación biométrica opcional para entornos exigentes.

Características

  • Fichaje móvil, tablet y web
  • Funciona offline con sincronización
  • Verificación biométrica opcional
  • Geolocalización del fichaje
  • Partes de trabajo digitales
  • Gestión de cuadrillas

Para qué sirve

  • Cumplir el registro horario
  • Imputar costes por parcela
  • Coordinar cuadrillas en tiempo real
  • Auditoría laboral

Integra con

  • Visual (operaciones)
  • Sistemas RR.HH. y nóminas
  • ERP (imputación de costes)
  • API (BI y reportes)
Pide demo de Selphi →
07 — Visual Sensor

Captura desde sensores e IoT.

Conexión con sensores físicos y virtuales para obtener datos de suelo, planta y ambiente en tiempo real. La red sensórica que alimenta los modelos predictivos de Agrovrain con datos hiperlocales de cada parcela.

Datos capturados

  • Humedad de suelo
  • Temperatura ambiente y de suelo
  • Conductividad y nutrientes
  • Pluviometría local
  • Datos de fertirrigación
  • Otros sensores compatibles

Para qué sirve

  • Decidir riego en tiempo real
  • Alimentar modelos de IA
  • Validar estado de cultivo
  • Reducir consumo de agua

Integra con

  • Sistemas de fertirrigación
  • Visual (parcela)
  • Agrovrain (modelos)
  • Maquinaria conectada
Pide demo de Visual Sensor →
08 — API

Integración bidireccional con ERP, CRM y sistemas externos.

La capa que abre el Ecosistema al stack tecnológico de cada cliente. API REST propia más conectores para ERP, CRM, BI, sistemas de maquinaria y bases de datos externas. El dato fluye en ambas direcciones sin doble entrada.

Conexiones disponibles

  • ERP, CRM y BI estándar
  • Sistemas de maquinaria (My John Deere y otros)
  • Bases de datos externas
  • Servicios OGC (WMS, WFS)
  • Importación de geometrías SIGPAC/SHP
  • Exportación a Excel, PDF, SHP

Para qué sirve

  • Conectar con tu stack actual
  • Evitar duplicar entradas de datos
  • Compartir datos campo↔planta
  • Soportar BI corporativo

Para quién

  • Departamentos IT corporativos
  • Integradores externos
  • Equipos de transformación digital
  • Partners tecnológicos
